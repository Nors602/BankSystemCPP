A simple console-based **Bank Management System** built using C++.  
The program manages client information and supports basic banking transactions through a menu-driven interface.

##  Features

###  Client Management
- Add new clients  
- Update client information  
- Delete clients  
- Search for a client  
- Display all clients in a formatted table  

###  Transactions
- Deposit money  
- Withdraw money with balance validation  
- View total balance of all clients  

###  File Handling
All client data is stored in a text file (`Clients.txt`) using a custom record format.

##  Technologies Used
- C++  
- File Handling  
- Structs & Vectors  
- Console UI  

##  How to Run
1. Open the project in Visual Studio (or any C++ IDE).  
2. Make sure `Clients.txt` exists in the project directory.  
3. Build and run the program.
